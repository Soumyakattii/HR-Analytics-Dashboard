# HR-Analytics-Dashboard

## 📌 Project Objective
This project focuses on analyzing employee workforce data using Power BI.  
The objective is to monitor employee demographics, promotion eligibility, service years, and workforce status through an interactive dashboard that supports HR decision-making.

---

##  Process
- Loaded the HR dataset into Power BI
- Cleaned and transformed the data using Power Query
- Created DAX measures for KPIs and percentage calculations
- Built interactive charts and KPI cards
- Designed a clean, user-friendly dashboard for HR insights

---

##  Questions Answered
- What is the total number of employees?
- What is the gender distribution of employees?
- How many employees are due for promotion?
- How many employees are not due for promotion?
- How many employees are currently in service?
- How many employees have been retrenched?
- What is the distribution of employees by service years?
- Which service year groups have the highest number of employees?

## 📈 Key Insights
- Total Employees: **1,470**
- Female Employees: **588 (40%)**
- Male Employees: **882 (60%)**
- Employees on Service: **1,353 (92%)**
- Retrenched Employees: **117 (8%)**
- Due for Promotion: **72 Employees**
